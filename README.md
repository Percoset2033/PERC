## ✨ Возможности

| | |
|---|---|
| 🎯 | **4 класса сложности** — от 1 до 4 класса |
| ⚡ | **Система энергии** — учит делать перерывы |
| 💡 | **Подсказки** — помогут решить трудный пример |
| 🪙 | **Магазин** — покупай подсказки и энергию |
| ⭐ | **Коллекция наклеек** — 15 штук для сбора |
| 🎵 | **Фоновая музыка** — спокойная мелодия |
| 💾 | **Автосохранение** — прогресс не пропадёт |
